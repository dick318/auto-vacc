# auto-vacc

自动预约深圳新冠疫苗

## 预览



## 说明

用户 token 需要自己去`深圳卫健委`公众号抓包拿到，如果要帮别人预约，又不方便抓他人手机上的包，可以通过自己手机给别人建档，然后抓包，抓`crmobile/registerUser/getUserInfo`这个接口的请求头 token，这个就是每个用户的 token
